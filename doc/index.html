1 What is MIDA?

MIDA is a minimalistic language for creating generative MIDI files. MIDA is not interactive in sense that you cannot control result of its activity in real time, it is indended for producers and should be used with DAW. MIDA can help you create variative elements in your music in very simple way. Since MIDI can control a lot of different instruments, power of MIDA is truly great.

2 Installation

It's absolutely possible to install MIDA on any platform for which Haskell Platform is supported. However, there are some precompiled options that may be more convenient and straightforward for some people. In particular, I propose installer for Windows users and self-extracting script for Linux users. To install MIDA on other platforms see section Compilation from Source.

2.1 Windows

To install MIDA on windows download MIDA installer, then run it. To uninstall MIDA run uninstaller.

2.2 Linux

Download self-extracting script and run it as root. To uninstall MIDA execute:

# mida-rem

This script will remove MIDA from your system.

2.3 Compilation from Source

First, install Haskell Platform. You also need to install Cabal. After that, download MIDA source code. Usually you need to update list of available packages:

$ cabal update
$ cabal install cabal-install

Now cd into directory containing downloaded repository of MIDA. Execute:

$ cabal configure
$ cabal build

It's very likely that Cabal will not compile MIDA from the first time so you'll need to install all dependencies that Cabal will list for you.

3 Invocation

MIDA can work as translator from source code into MIDI files, and also as interactive environment (interpreter). Here's invocation scheme:

mida [-i|--interactive] [-s|--seed SEED] [-q|--quarter TICKS] [-b|--bars BARS] [-o|--output OUT] [FILE]

Let's list all available options:

OptionDescription
-h,--helpShows this information about invocation parameters and available options
-i,--interactiveEnables interactive (interpreter) mode.
-s,--seedSets seed for MIDI generation, default value is 0
-q,--quarterSets number of ticks for quarter note, default value is 24
-b,--barsSets total time of result MIDI file as number of quarter notes (bars), default value is 16
-o,--outputSpecifies output filename. Default value is evaluated as FILE with extension .mid

If you don't supply any file as command line argument, MIDA will run in interactive mode even without -i option. Options -s, -q, and -b are ignored if option -i is used.

4 Configuration File

One can customize MIDA to some extent using configuration file. One should create the file in his home directory. This directory will be different on different operating systems. On Linux machines it would be something like /home/username/.mida, while Windows users should create the file in C:\Users\username\.mida.

For now, only two parameters can be changed:

  • • prompt (as prompt);
  • • length of displayed result of evaluation (as length).

Syntax of .mida file is one of Unix configuration file. That is, you can use # to start comments, and = to assign new values (spaces before and after = are allowed).

Here is complete example of .mida file:

#
# MIDA Configuration File
#

prompt = "M> "
length = 20

5 MIDA Tutorial

Now that you know all basic information about interpreter we can start our journey into magic world of MIDA.

MIDA is minimalistic, but not very flexible or complex. In fact, it is so simple that any statement that has valid synax will produce valid result. A MIDA statement only has to pass parser, that's all. In fact, it's quite hard to enter invalid statement in MIDA and we'll see why.

If you're interested in generative music, you probably know that there are lots of really advanced programs for algorithmic composition. But take a look at output of those programs. I don't mean to offend someone, but it's often not that impressive.

What is the reason of that? Well, people who work with complicated tools tend to pay more attention to process than result. The more compilated tool you use the more time you spend on tool itself. Algorithm may be really impressive, but audience just don't give a fuck. We're gonna use basic tools of MIDA to create super sexy pop stuff, that's what we need, man, we need to be primitive and successful.
